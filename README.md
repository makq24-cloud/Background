# DTG Background Removal

Companion tool to TeeSort for Josh's t-shirt printing workflow.

**Live:** https://makq24-cloud.github.io/background/

Browser-based tool: import a graphic, crop, place top-aligned on a transparent
DTG platen template, remove the black or white background via color-to-alpha,
touch up with restore/erase brushes, export as transparent PNG.

Source for this build lives outside the repo. The files in `main` are the
built static site served by GitHub Pages from the repo root.
